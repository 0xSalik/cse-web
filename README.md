# CSE Department Website Redesign (IUST)

This repository contains a project to redesign and redevelop the CSE department website at the Islamic University of Science and Technology (IUST).

## Objective
To modernize the website with improved UI/UX, better accessibility, and a clean, responsive design while maintaining clarity and usability for students, faculty, and visitors.

## Scope
- UI/UX redesign
- Responsive frontend implementation
- Content restructuring
- Performance and accessibility improvements
- Revision of faculty profiles
- Improvements to the student portal
